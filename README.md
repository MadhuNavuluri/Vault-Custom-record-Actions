# Vault-Custom-record-Actions
used for vault CRM custom record action
Deploy VPK: Admin → Deployment → Inbound Packages → Import → Review & Deploy
Assign Action to Object: Admin → Configuration → Objects → select object → Actions Tab → Create → Select Action
Enable on Object Type: Admin → Configuration → Objects → select object → Object Types → Edit Object Type Actions → Check ✅ the action
Grant Permission: Admin → Users & Groups → Permission Sets → [Profile] → elect object → Object Action Permissions → grant  → View ✅ Execute ✅
